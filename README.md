How to Run Project
❖ Install NodeJs 
1. Visit the official Node.js website i.e) https://nodejs.org/en/download/ 
2. Download the Node.js installer 
3. Run the installer.
                      — First Run Server then Client — 
❖ Steps to setup Server of the project 
1. Open Project Folder In VS Code 
2. Setup The MongoDB & obtain MongoURI 
      Link : https://www.mongodb.com/cloud/atlas/register
3. Setup Cloudinary 
Now Open Server Folder in Integrated Terminal & use Below Commands 
   npm install  
   npm run server  
Now Open Client Folder in Integrated Terminal & use Below Commands 
     npm install  
     npm run dev
4. Setup Stripe  
    Link : https://dashboard.stripe.com/login 
5. Push Project to Github & Deploy Backend on Vercel 
    Link : https://vercel.com/  
6. Add Backend Live URL in Stripe Webhook 
   a. Obtain Stripe Signing Key 
   b. Add the key in env 
7. Deploy Frontend  
   a. Deploy Frontend on Vercel
   b. Add Frontend URL as Origin
   c. Then redeploy the backend project
